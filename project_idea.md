# Project Concept: SkipTube (TubeCapsule)

## Overview
SkipTube (also known as TubeCapsule) aims to revolutionize how users interact with educational and informational video content. Moving beyond traditional video viewing experiences, this platform eliminates unnecessary navigation and provides highly interactive, AI-driven learning tools right alongside the video.

## Core Features

### 1. Same-Language Smart Notes (With Code/Key Terms)
- **Contextual Language Support**: Notes are generated in the natural language of the video (e.g., Hindi/Hinglish for Hindi videos) rather than forcing pure English translations.
- **Rich Markdown Formatting**: Clean markdown notes with bolded key terms and distinctly formatted code blocks for technical content.

### 2. Click-to-Jump Interactive Timestamps
- **Seamless Navigation**: Each generated note or bullet point includes a clickable timestamp (e.g., `[12:34]`).
- **Auto-Skip Integration**: Clicking a timestamp automatically seeks the video player to that exact moment, eliminating the need for manual timeline scrubbing.

### 3. Distraction-Free Contextual Suggestions
- **Targeted Recommendations**: Bypasses YouTube's standard algorithm (which often pushes clickbait) to suggest exactly 3 carefully curated videos.
- **Knowledge Gap Bridging**: Recommendations are AI-analyzed to fill specific knowledge gaps related to the current video's topic.

### 4. Ask the Video (Chat Interaction)
- **Conversational AI**: A built-in chatbox allows users to query the video directly (e.g., "What code was written at the 15th minute?").

### 5. 1-Click Export
- **Portability**: Users can export their notes instantly using a 1-click Markdown copy feature or directly integrate them into Notion.

### 6. Interactive Dashboard (History Log)
- **User Authentication**: Integrated with Supabase Auth for secure login.
- **Personal Library**: A dedicated dashboard where users can access and manage their history of summarized and annotated videos.
